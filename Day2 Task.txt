5-  Git Rebase:
A command that integrates changes from one branch into another by rewriting the commit history. 
 
 how to use it  : 1 - Switch to the branch you want to rebase:     git checkout feature-branch
                  
                  2- Rebase onto another branch:  git rebase main

                  3- Resolve conflicts and continue: git rebase --continue
 
                  4-Push the rebased branch  :git push --force
